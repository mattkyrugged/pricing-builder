# Partner Pricing Builder

Internal tool for **Kentucky Rugged Technology Solutions** to create and manage customizable partner pricing reference sheets.

## Features
- **Product Catalog** — Import products from Zoho Books CSV, manage SKUs and pricing
- **Sheet Builder** — Create branded partner pricing sheets with live preview
- **PDF Export** — Print-ready output for partners
- **Cloud Saved** — All sheets persist in Supabase

## Tech Stack
- React + Vite
- Supabase (PostgreSQL + API)
- Vercel hosting

## Setup

```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → auto-deploys via Vercel.
